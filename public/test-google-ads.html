Google Ads Integration Test

Use this page to test your Google Ads OAuth configuration and API connection

Prerequisites Checklist:

  • GOOGLE_ADS_CLIENT_ID secret configured in Supabase
  • GOOGLE_ADS_CLIENT_SECRET secret configured in Supabase
  • GOOGLE_ADS_DEVELOPER_TOKEN secret configured in Supabase
  • Redirect URI added to Google Cloud Console OAuth client
  • You are logged into the Revoa app

Test 1: Check Secrets Configuration

Verify that the required environment variables are configured in your Edge Functions.

Test 2: Start OAuth Flow

This will open Google's OAuth consent screen in a popup window.

Test 3: Check Connected Accounts

View all Google Ads accounts connected to your user.
